## 💬 Chat Online em Tempo Real

Aplicação web de chat em tempo real desenvolvida com foco em comunicação instantânea entre usuários, utilizando tecnologias modernas e arquitetura baseada em eventos.

---

## 🧠 Sobre o Projeto

Este projeto foi desenvolvido com o objetivo de praticar e demonstrar conceitos de comunicação em tempo real utilizando WebSockets, além de reforçar conhecimentos em desenvolvimento web.

A aplicação permite que múltiplos usuários se conectem e troquem mensagens instantaneamente, simulando um ambiente de chat moderno.

---

## ⚙️ Funcionalidades

* 💬 Envio e recebimento de mensagens em tempo real
* 👥 Comunicação entre múltiplos usuários
* 🔄 Atualização automática sem recarregar a página
* 🌐 Interface web simples e interativa
* ⚡ Comunicação baseada em eventos (WebSockets)

---

## 🛠️ Tecnologias Utilizadas

* JavaScript
* WebSockets
* HTML / CSS
* Node.js

---

## ▶️ Como Executar

```bash
git clone https://github.com/allan141/Chat-Online.git
cd Chat-Online
npm install
npm start
```
